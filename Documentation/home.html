SCH is a modern high-level general purpose programming language. Its goal is to be fast, but still intuitive and practical. It features classes, null safety, detailed error messages, automatic memory management, and a simple and modern syntax.

Features

Memory management:
SCH is compiled but there are no memory leaks or life time problems like in bad C code or complicated memory managing systems like in Rust. Instead, it uses shared pointers to give the user a Java/C#-like experience. Of course, this slightly reduces the efficiency, but it's worth it.

Error messages:
Error messages always tell you exactly where the error occured, even at runtime.

Smart casting:
SCH features smart casting of classes and nullables in most simple cases.
